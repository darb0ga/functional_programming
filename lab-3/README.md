### Ключевые элементы реализации

- `algorithms/linear-step` — потоковая линейная интерполяция отрезками между соседними узлами  
- `algorithms/newton-step` — потоковая интерполяция методом Ньютона по окну из `n` точек  
- `algorithms/newton-coeffs` / `algorithms/newton-eval` — вычисление коэффициентов полинома Ньютона и значения полинома в заданной точке  
- `algorithms/build-algorithms` — формирование набора активных алгоритмов в зависимости от аргументов командной строки  

- `core/parse-args` — разбор аргументов командной строки (`--linear`, `--newton`, `--step`, `-n`)  
- `core/parse-point` — парсинг входных строк формата `x y`, `x;y`, `x,y`, `x<TAB>y` в структуру `{:x .. :y ..}`  
- `core/process-stream` — потоковая обработка stdin: при поступлении каждой точки вызывает алгоритмы, выводит новые значения и завершает работу по EOF или пустой строке  
- `core/print-point` — форматированный вывод интерполированных точек (фиксированное число знаков после запятой)

---

## Запуск и аргументы

Поддерживаются флаги:

- `--linear` — линейная интерполяция;
- `--newton` — интерполяция Ньютона;
- `--step h` — шаг дискретизации результирующих данных;
- `-n k` / `--n k` — число точек в окне для Ньютона.


## Выводы

В ходе лабораторной работы было разработано потоковое приложение для интерполяции таблично заданной функции на языке Clojure. Реализованы два алгоритма: линейная интерполяция отрезками и интерполяция методом Ньютона по окну заданного размера.  

Приложение соответствует требованиям: алгоритмы оформлены в функциональном стиле, настройки работы (выбор алгоритмов, шаг дискретизации, размер окна) задаются через аргументы командной строки, а обработка входных данных осуществляется в потоковом режиме.
